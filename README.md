# bleta web framework with [golang](https://golang.org)

### Feature Overview 

1- MVC Architecture 
2- Template rendering
3- Static Files 
4- URL mapping
5- Validation
6- Extensible middleware framework
7- support ORM (GORM)
8- Extensible Database (MySQL, ...)
9- Extensible Security (CSRF , Clickjacking , Secure Header and etc )
